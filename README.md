# 🗺️ TransportMap — Grand Abidjan

> Cartographie interactive du réseau de transport urbain du District Autonome d'Abidjan

## 📋 Description

Application web de cartographie interactive permettant de visualiser et analyser le réseau de transport en commun du Grand Abidjan. Développée dans le cadre d'un projet d'expertise SIG pour l'AMUGA.

### Fonctionnalités

- Carte plein écran avec fond CARTO Dark Matter
- 7 lignes de transport (bus express, standard, suburbain, lagunaire)
- 8 hubs de transport majeurs avec informations détaillées
- 13 communes avec données démographiques
- Projet Métro Ligne 1 (Anyama → Aéroport FHB) en tracé projeté
- Panneaux glass morphism flottants
- HUD tactique avec coordonnées en temps réel
- Interactions : clic sur lignes/hubs, fly-to, filtrage de couches

## 🚀 Installation

```bash
git clone https://github.com/edgar-kouassi/transport-abidjan.git
cd transport-abidjan
npm install
npm run dev
```

## 🛠️ Stack

React 19 + Vite + Leaflet + react-leaflet + Lucide Icons
